# Translating parser for Prance #

## Requirements

* pipenv
  ```sh
  $ pip install pipenv
  ```

## Development environment ##

```sh
$ pipenv sync
```

## Testing ##

```sh
$ PYTHONPATH=. pipenv run pytest
```
